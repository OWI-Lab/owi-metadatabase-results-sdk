# How-to Guides

How-to guides are **problem-oriented** recipes. Each one shows you how to
accomplish a specific task with the Results SDK.

## Setup

| Guide | Problem it solves |
|-------|-------------------|
| [Install the SDK](install.md) | Get the package installed via pip or uv. |
| [Authenticate](authenticate.md) | Configure API token authentication. |

## Working with Data

| Guide | Problem it solves |
|-------|-------------------|
| [Upload Results](upload-results.md) | Persist analyses and result rows to the backend. |
| [Query Results](query-results.md) | Retrieve and filter result data through the SDK. |
| [Plot Results](plot-results.md) | Render interactive charts from persisted results. |
| [Filter Data](filter-data.md) | Use Django-style lookups to narrow query results. |
