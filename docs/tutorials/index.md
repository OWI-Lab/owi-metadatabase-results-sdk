# Tutorials

Tutorials are **learning-oriented** lessons. Each one guides you through a
complete workflow so you gain hands-on experience with the Results SDK.

| Tutorial | What you will learn |
|----------|---------------------|
| [Your First Results Query](first-query.md) | Connect to the API, list analyses, and retrieve result rows. |
| [Lifetime Design Frequencies Workflow](lifetime-design-frequencies.md) | Prepare workbook data, upload frequency results, retrieve and plot them. |
| [Lifetime Design Verification Workflow](lifetime-design-verification.md) | Upload verification-frequency results with timestamp control. |
| [Wind Speed Histogram Workflow](wind-speed-histogram.md) | Parse histogram bins, upload scoped series, and render histogram plots. |
| [CEIT Corrosion Monitoring Workflow](ceit-corrosion-monitoring.md) | Load CEIT JSON data, match SHM sensors, upload and plot corrosion metrics. |
